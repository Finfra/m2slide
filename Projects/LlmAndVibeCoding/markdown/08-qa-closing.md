## 도구 선택 관련

**Q1: 처음 시작하는 개발자에게 어떤 도구를 추천하나요?**

**추천**: 1세대 (Cursor)

**이유**:
* GUI 환경이 직관적하고 학습 곡선이 낮음
* 시각적 피드백으로 변경사항을 즉시 확인 가능
* VSCode와 유사한 인터페이스로 친숙함

**상세 학습 경로는 5장 참조**

---

## Q2: Cursor와 Claude Code 중 선택

**비교 기준**

| 기준 | Cursor 추천 | Claude Code 추천 |
|------|------------|-----------------|
| 터미널 경험 | 없음/초보 | 능숙함 |
| 주요 작업 | 프론트엔드 | 백엔드/인프라 |
| 프로젝트 수 | 1-2개 집중 | 다중 프로젝트 |
| 로컬 리소스 | 충분 (8GB+) | 제한적 |
| 자동화 필요 | 낮음 | 높음 |

---

## Q2: 권장사항

**두 도구를 모두 사용하는 것도 좋은 전략**

```
프론트엔드 작업: Cursor (UI 프리뷰 필요)
백엔드/인프라: Claude Code (자동화 효율적)
테스트/배포: Claude Code (스크립트 자동화)
```

## Q3: 무료로 시작할 수 있는 도구

**무료 옵션**

**Cline (VSCode 확장)**
* 완전 무료 오픈소스
* 다양한 LLM 모델 선택 가능
* 자체 API 키 사용 시 비용만 지불

**ChatGPT 무료 버전**
* 기본적인 코드 생성 가능
* 복사-붙여넣기 방식
* 제한적이지만 학습용으로 적합

```

❌ 생성된 코드를 이해하지 않고 사용
❌ 기본 개념 학습 생략
❌ 디버깅 능력 상실
```

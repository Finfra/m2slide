jobs:
  ai-review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: AI Code Review
        run: |
          claude << EOF
          코드 품질 분석해줘
          보안 취약점 찾아줘

## 주의사항

**피해야 할 것**

❌ **맹신하지 않기**
* AI 코드를 무조건 신뢰하지 말 것
* 항상 코드 리뷰 필수
* 보안 취약점 수동 검증

❌ **과도한 의존**
* 기본 코딩 스킬 유지
* 알고리즘 이해도 중요
* 문제 해결 능력 계속 연마

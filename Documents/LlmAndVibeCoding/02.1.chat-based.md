## 0세대: 작동 방식

**워크플로우**
```
1. 개발자 → 자연어로 질문
2. LLM → 코드 스니펫 생성
3. 개발자 → 복사하여 IDE에 붙여넣기
4. 수정 필요 → 다시 채팅으로 질문
```

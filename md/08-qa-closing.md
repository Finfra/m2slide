# 8. 자주 묻는 질문 (Q&A)

---

## 도구 선택 관련

**Q1: 처음 시작하는 개발자에게 어떤 도구를 추천하나요?**

**추천**: 1세대 (Cursor)

**이유**:
- GUI 환경이 직관적하고 학습 곡선이 낮음
- 시각적 피드백으로 변경사항을 즉시 확인 가능
- VSCode와 유사한 인터페이스로 친숙함

---

## Q1: 학습 경로

**Week 1-2**: Cursor 기본 기능 익히기
- Cmd/Ctrl + K로 간단한 요청
- diff 모드로 변경사항 확인
- 작은 컴포넌트 생성 연습

**Week 3-4**: 실전 프로젝트 적용
- 사이드 프로젝트에 활용
- 코드 리뷰 습관 형성
- 베스트 프랙티스 학습

---

## Q2: Cursor와 Claude Code 중 선택

**비교 기준**

| 기준 | Cursor 추천 | Claude Code 추천 |
|------|------------|-----------------|
| 터미널 경험 | 없음/초보 | 능숙함 |
| 주요 작업 | 프론트엔드 | 백엔드/인프라 |
| 프로젝트 수 | 1-2개 집중 | 다중 프로젝트 |
| 로컬 리소스 | 충분 (8GB+) | 제한적 |
| 자동화 필요 | 낮음 | 높음 |

---

## Q2: 권장사항

**두 도구를 모두 사용하는 것도 좋은 전략**

```
프론트엔드 작업: Cursor (UI 프리뷰 필요)
백엔드/인프라: Claude Code (자동화 효율적)
테스트/배포: Claude Code (스크립트 자동화)
```

## Q3: 무료로 시작할 수 있는 도구

```

절감 비용: 60시간 × $50 = $3,000/월

도구 비용: $20/월
순이익: $2,980/월

ROI: 14,900%
```
